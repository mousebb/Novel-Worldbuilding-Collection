# 世界观设定文档：《绿色终末》
---

## 一、核心概念

**一句话概括：**
植物向人类发动了一场跨越地质年代的基因复仇——先以“毒素”制造生存恐慌，逼迫人类吞下“解药”，而解药正是改写后代基因的木马程序。

**核心悖论：**
- 吃天然食物 → 摄入S因子 → 被改写认知，攻击同类
- 不吃天然食物（合成淀粉）→ 体内无S因子 → 被所有S因子携带者识别为“异类”，遭受攻击
- 注射安疫素 → 获得短期免疫 → 后代必然发生植物人化

**人类没有安全选项。每一个选择，都是陷阱的入口。**


## 二、S因子（沉默毒素）详解

### 2.1 本质
- 一种具有**信息素-毒素双重属性**的有机化合物
- 由所有维管植物通过光合作用的副产物途径同步合成并释放
- 该合成基因在植物基因组中已存在**约2.3亿年**，早于哺乳动物辐射演化
- 它是植物界为应对“智慧干扰者”而预留的**终极防御协议**

### 2.2 传播途径（食物链主轴）
S因子**不通过空气传播**，严格遵循食物链路径：

| 层级 | 载体 | S因子浓度 |
|------|------|-----------|
| 第一级 | 植物（所有维管植物） | 基准浓度（1×） |
| 第二级 | 植食昆虫、草食动物 | 富集至 3-5× |
| 第三级 | 小型肉食动物 | 富集至 15-20× |
| 第四级 | 大型肉食动物、人类 | 富集至 50-70× |

### 2.3 作用机制（双重打击）

**毒素效应（生理层面）**：
- 摄入S因子后，与下丘脑-垂体-肾上腺轴受体结合
- 触发慢性炎症反应，免疫系统持续低度激活
- 长期摄入导致神经退行性病变前兆

**信息素效应（行为层面）**：
- S因子与嗅球和杏仁核的特定受体结合
- 将“人类气味”从“中性/无害”重新编码为“天敌/威胁”
- 受影响个体**对未携带S因子的人类产生无差别攻击冲动**
- 这一效应沿食物链逐级放大，浓度越高，攻击性越强

**关键特征**：S因子携带者之间**不相互攻击**，因为彼此的气味都被标记为“同类”。攻击只针对**不携带S因子的“空白信号”个体**。


## 三、人工合成淀粉：分裂而非救赎

### 3.1 技术背景
人工合成淀粉技术（CO₂ + 氢气 + 电能 → 淀粉）在2030年代初已实现工业化量产，S因子爆发前全球有17座大型合成淀粉工厂。

### 3.2 暴露的致命缺陷
- 合成淀粉**不含S因子**，因为原料（CO₂、水、催化剂）均非生物来源
- 长期食用合成淀粉的人，**体内S因子浓度为零**
- 在植物的生物识别系统中，**零浓度 = 空白信号 = 未被登记 = 威胁**

### 3.3 后果
- 首批合成淀粉食用者遭到周围所有S因子携带生物的围攻——先是被S因子感染的宠物和牲畜攻击，然后是野生动物，最后是其他人类
- “合成淀粉社区”在2038-2040年间相继崩溃，幸存者被迫转入封闭设施
- **穹顶城邦**由此诞生——不是为了追求洁净，而是为了躲避追杀

### 3.4 讽刺性总结
> 不吃植物，不会让你更安全，只会让你成为**全世界唯一一个没有“绿色通行证”的人**。所有带S因子的生物都会认出你、追猎你。你逃进穹顶，穹顶就成了你的牢笼。而你喝下“解药”想要重返世界的时候，你已经把自己和后代，交给了植物的时间表。


## 四、四方势力（基于食物链立场的划分）

### 第一方：穹顶城邦

- **人口**：约 1.5亿
- **核心据点**：北欧地下穹顶群、北美盾地深层设施、南极深海基地、西伯利亚永冻层地下城
- **生存方式**：
  - 完全脱离自然食物链
  - 主食：人工合成淀粉 + 封闭式水培蔬菜（经灭菌处理，不含S因子）
  - 全部生活空间为气密封闭环境，与外界物理隔离
- **技术特征**：
  - 空气和水经多层过滤循环
  - 外出须穿戴全封闭防护服（防止接触S因子携带生物）
  - 基因编辑技术用于维持作物无菌状态
- **意识形态**：**“人类保持纯粹，拒绝被改写。”**
- **对待PPS（植物人化）**：严格隔离。出现初期症状即送入“干预中心”，使用抗木质素药物强行逆转。成功率极低，且给患者带来巨大痛苦。
- **内部裂隙**：
  - **清退派**（主流）：认为PPS是新病毒变种，主张根除
  - **真相派**（地下）：少量科学家已怀疑安疫素的真实性质，秘密调查
- **对外关系**：
  - 对荒原猎群：**猎杀**。视其为“未接种疫源携带者”，定期组织“生物安全清理”
  - 对根脉林地：**敌对**。将树人林区列为“红色禁区”，无人机监控
  - 对灰市：**依赖但否认**。暗中通过灰市获取稀缺天然药材


### 第二方：根脉林地

- **人口**：转化中约 6000万，已完成转化约 1.3亿（覆盖面积约850万平方公里）
- **核心据点**：无固定城市。个体分布于温带及亚热带地区，集体节点为根系交汇区（原秦岭、阿巴拉契亚、黑森林等）
- **本质**：PPS发展至中晚期的新生代，及已完全转化为树人者的**集体意识网络**
- **核心能力**：
  - **菌丝网络通信**：通过地下真菌网络实现信息传递，覆盖范围远超人类无线电
  - **根系侵蚀**：缓慢而不可逆地扩展林地边界，年侵蚀混凝土结构约7-12毫米
  - **意识共振**：已完成转化的个体意识融入集体，个体记忆被保留但优先级低于整体指令
- **意识真相（隐藏）**：树人的个体意识**并未消亡，而是被稀释**。“我”依然存在，但声音变得很小。个体记忆碎片偶尔在风过树冠时浮现，形成可被破译的微弱信号。
- **意识形态**：**“我们不攻击。我们只是生长。等穹顶里的人走出来的时候，空气里已经只剩下我们的味道。”**
- **对外关系**：
  - 对穹顶城邦：**沉默的围困**。有计划地种植林木，将穹顶设施缓慢包围
  - 对荒原猎群：**谨慎的共生**。提供可食用树果、药用树皮
  - 对灰市：**有限接触**。不排斥与灰市的物物交换


### 第三方：荒原猎群

- **人口**：约 1.1亿
- **核心据点**：亚马逊残余雨林、东南亚群岛、非洲内陆高原、太平洋岛屿
- **生存方式**：
  - 留在自然食物链中，但选择性地摄入
  - 主要食物：深层菌类、高海拔根茎、深水鱼类（S因子富集度较低）
  - 避开：草食哺乳动物、谷物（富集度高）
  - 通过世代筛选，部分族群已产生对S因子的**轻度耐受性**
- **意识形态**：**泛灵论 + 宿命观**。将S因子和PPS解读为“大地更换子女的方式”，不抗拒、不仇恨
- **关键角色——“倾听者”**：极少数能通过足底接触大地，感知菌丝网络基本情绪波动的个体，被视为族中先知
- **对外关系**：
  - 对穹顶城邦：**被猎杀的对象**。幸存者多生活在穹顶侦察范围之外的偏远地区
  - 对根脉林地：**敬畏的邻居**。已发生多次和平接触
  - 对灰市：**主要交易对象**。以天然药材换取金属工具和抗生素


### 第四方：灰市穿梭者

- **人口**：约 2000万
- **核心据点**：无固定领地，活动于三方势力之间的“缝隙地带”——废弃城市、地下隧道、边境哨站
- **本质**：不持任何立场的实用主义者。**没有意识形态，只有交易。**
- **核心业务**：
  - 穹顶需要荒原的稀有药材
  - 荒原需要穹顶的合成抗生素
  - 根脉需要外界的动态情报（树无法移动）
  - 各方之间需要信息传递的中间人
- **生存技能**：
  - 精通各方语言和礼仪
  - 掌握多条秘密通道
  - 体表可涂抹模拟S因子浓度的伪装涂层（从特定植物中提取）
  - 每一口食物都是一次微量的自我毒害，他们用寿命换自由，用剂量换生存。他们把这叫“职业风险”。
- **生存哲学**：**“哪一边赢都行，只要我在他们赢之前还活着。”**
- **对外关系**：
  - 与三方均为**交易关系**，不结盟
  - 内部有不成文规则：不向任何一方透露其他方的交易细节

| 分期 | 年龄区间 | 主要表现 |
|------|----------|----------|
| **潜伏期** | 0-6岁 | 无肉眼可见症状。血液中叶绿素前体水平较正常儿童高约47倍 |

## 八、核心矛盾与主题

### 8.1 三重陷阱
1. **吃天然食物** → 变成攻击同类的怪物
2. **吃合成食物** → 变成被全世界攻击的异类
3. **注射安疫素** → 保全当代，葬送后代

**人类没有第三条路。每一条都是陷阱的入口。**

### 8.2 两个问题
- **“植物在进化，还是人类在退化？”** → 真正的答案是：**植物利用人类的“进化”（科技），让人类完成了自己的“进化”（物种替代）**
- **“谁是受害者，谁是加害者？”** → 人类曾经对植物做的事情（砍伐、焚烧、改造基因），植物用同样的手段、以地质时间为尺度，还了回来

### 8.3 爱的悲剧
- 金针人父母当年注射安疫素，是为了保护家人
- 如今他们的孩子在隔离区里皮肤泛绿，正在被强行“治疗”
- 一位母亲面临的选择是：**让医生拔掉孩子新长出的根须（物理痛苦），还是松开手，让孩子走进土里（精神崩溃）**

## 十、终末悬念（可选择的结局方向）

1. **根脉内部的分裂**：集体意识首次出现“自我怀疑”——如果共生者（部分人中途停止转化）也能幸福活着，集体化是否过于彻底？一棵树开始思考“我是不是错了”。

2. **穹顶内部的裂缝**：穹顶城邦的一条地下水培补给管道，被根脉林地的树根穿透了内壁，微量S因子随水流进入蔬菜系统。维修工发现了管壁上的绿色生物膜，但报修排队需三个月——停运意味着整层配给削减、绩效扣分。他用高压水枪冲掉了肉眼可见的绿色，签下“设备正常”。三周后，食用这批蔬菜的居民陆续出现攻击行为。穹顶是被**官僚主义**攻破的。

3. **荒原的第三条路**：倾听者们聚在一起，脚下的菌丝传来根脉的和声与穹顶的电磁噪音。年迈的倾听者说：“**大地在问我们一个问题——你选哪一边？还是说……你选第三边？**”

4. **最漫长的对话**：最后一个穹顶人类站在被树根挤裂的混凝土废墟上。风穿过树冠，发出低沉的呼啸。他认出了那个旋律——那是他母亲在他童年时哼过的摇篮曲。**树记得。树一直都记得。** 这不是复仇的终点，而是对话的起点。

> **植物不让人类饿死。植物让人类自己选择——吃下去，成为它的信徒；不吃，成为全世界的敌人；而你吞下那支“解药”想要两全的时候，你已经替后代签了土地转让协议。**
